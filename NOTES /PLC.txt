IF POD CRASH : kubectl logs mypod

| RestartPolicy | क्या होता है जब container fail होता है? |
| ------------- | --------------------------------------- |
| Always        | हमेशा दोबारा चलाया जाएगा (default)      |
| OnFailure     | सिर्फ़ तब चलेगा जब exit code ≠ 0 हो     |
| Never         | कभी नहीं चलेगा                          |

Deployment हमेशा RestartPolicy: Always रखता है।
